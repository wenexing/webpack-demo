[指南](https://doc.webpack-china.org/guides/)
## 全局安装
```
npm install --global webpack
```

## 起步
```
mkdir webpack-demo && cd webpack-demo
npm init -y
npm install --save-dev webpack

npm install --save lodash

webpack src/index.js dist/bundle.js
```

NPM 脚本(NPM Scripts)
```
npm run build
```
